# typescript-plugin-styled-components SSR example

## Getting Started

- Build `typescript-plugin-styled-components`

```
$ cd .. 
```

```
$ npm i
```

```
$ npm run build
```

- Run example 

```
$ cd example 
```

```
$ npm i
```

```
$ npm run dev
```

Note: see  [`ttypescript`](https://github.com/cevek/ttypescript) if you want to compile the server without webpack
